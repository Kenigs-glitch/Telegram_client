Telegram multi-account (portable via Docker)

Требования:
- Docker и Docker Compose
- X11 (Linux desktop)

Установка и запуск:
1) В каталоге проекта создайте папки `tdata1`, `tdata2`, ... (скопированные папки из разных телеграм-профайлов)
2) Запустите:

```bash
chmod +x ./run.sh
./run.sh
```

3) Выберите нужную `tdata*`. Контейнер запустится.
4) Для отправки файлов кладите их в папку `shared` в каталоге проекта. Внутри контейнера эта папка доступна по тому же абсолютному пути, что и на хосте.
5) Команды в меню:
   - `l` — логи
   - `s` — остановить контейнер
   - `r` — перезапустить с текущим tdata
   - `c` — переключить tdata (down → выбор → up)
   - `q` — выйти из скрипта (контейнер продолжит работать)

Источник бинарного архива Telegram: https://telegram.org/dl/desktop/linux
